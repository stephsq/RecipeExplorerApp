**Recipe Explorer App Description:**

This is an Android application built using Kotlin with Jetpack Compose. This app allows you to explore different recipes and view details.

**Features:**

- recipe browsing with scrollable list
- detail page of recipe

**How to Run:**

  1. Clone the Repository:
     - git clone https://github.com/stephsq/RecipeExplorerApp.git
  2. Open in Android Studio:
      - Open Android Studio.
      - Select "Open" and navigate to the cloned project folder.

**Build & Run:**

  1. Wait for Gradle to finish syncing.
  2. Select an Emulator or a physical device.
  3. Click the Run button (green play icon).
